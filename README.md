# Dignam.xyz

> A base page to direct visitors to a desired site.

## Usage

Install

```bash
git clone https://github.com/sbdchd/sbdchd.github.io && sbdchd.github.io
npm install
npm install -g grunt-cli
```

Build

```bash
grunt clean
grunt
```

Develop

```bash
grunt watch
# in another shell
grunt connect
```
